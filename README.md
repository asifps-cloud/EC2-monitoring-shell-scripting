# EC2 Monitoring Script

##Description

A Bash script that monitors an Ubuntu EC2 instance and displays its services, CPU usage, memory usage and disk usage.

##Features

- Displays hostname
- Shows uptime
- CPU usage
- Memory usage
- Disk usage
- Services

##Automation

Configured using cron to execute daily at 2:30 pm. 

##Technologies

- Ubuntu EC2
- Bash
- Git
- GitHub
